# 🚀 Digitools Platform

## 📌 Overview

**Digitools Platform** is a modern web application designed to provide users with powerful digital tools in one place. It offers a clean interface, smooth interactions, and efficient features to enhance productivity and user experience.

---

## 🛠️ Technologies Used

- ⚛️ React.js
- 🎨 Tailwind CSS
- 🌼 DaisyUI
- 📜 ES6 JavaScript
- 🗂️ JSON
- 🔔 React Toastify

---

## ✨ Features

- ⚡ **Dynamic UI Components**
  Responsive and reusable components built with React and styled using Tailwind & DaisyUI.

- 🛒 **Interactive Pricing & Cart System**
  Users can explore plans, add items to cart, and manage selections smoothly.

- 🔔 **Real-time Notifications**
  Instant feedback using React Toastify for actions like adding items or errors.
